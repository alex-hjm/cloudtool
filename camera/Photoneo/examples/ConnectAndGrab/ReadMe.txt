========================================================================
    CONSOLE APPLICATION : ConnectAndGrab Project Overview
========================================================================

This is a simple example of how to find and connect an available PhoXi Device.

You will learn how to:

* check if PhoXi Control is running,
* obtain a list of devices connected to the network,
* connect to a device attached to PhoXi Control if there is one, or to
  the last device on the list if there is not,
* different ways to capture scans.
