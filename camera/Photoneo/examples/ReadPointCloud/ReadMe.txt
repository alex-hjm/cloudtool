========================================================================
    CONSOLE APPLICATION : ReadPointCloudExample Project Overview
========================================================================

Console example application which after connecting to device and triggering
a scan, shows multiple ways how to handle returned point cloud data with
Photoneo PhoXi API.

You will learn how to:

* connect to a device from a list of devices connected to the network
  by requesting its ID,
* work with the acquired data and save it into a specific data structure,
* save scans into different file formats.
